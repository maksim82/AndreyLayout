Осиновая стружка Aspen World

Стружка осиновая используется в качестве подстилки для животных. Стружка не содержит примеси коры, пыли и посторонних предметов. Для удобства хранения стружка упакована в плотные брикеты, которые выполненые из прозрачной пленки, для того чтобы покупатель видел качество товара. Вес одной упаковки (брикета) составляет 550 гр. Приблизительный объём 20л.
